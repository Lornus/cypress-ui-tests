# Test stack: 
1. node: v14.17.0
2. Cypress package version: 7.6.0
3. MochaJs
4. Allure reporter
5. chai

# For running test 
```
git clone 
```
```
npm install 
```
### Launching test in cypress studio
```
npm run cy-studio:test 
```
### Launching test in terminal 
```
npm run cl:test  
```
### Allure report
```
npm run report
```

# Test stack: 
1. node: v14.17.0
2. Cypress package version: 7.6.0
3. mochaJs
4. Allure reporter
5. chai


# For running test 
1. clone repo 
2. install node if you don't have 
3. install cypress if you don't have
4. npm install && npm cit
5. npm test 
6. to see reporter(allure html): npm run report
